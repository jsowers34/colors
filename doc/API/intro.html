Introduction to colors

At the Beginning of Time (January 1st, 1970 at 00:00:00 UTC) - at least for UNIX, there was a defined color map relating color names to RGB values. For example, Antique White" was defined as :red 250, green 235, blue 215 and Light Goldenrod was red 238, green 221, blue 130.

There were some 900 colors defined in the original RGB table - many redundant

Despite the size and redundancy, there were many conveniences in referring to a color name rather than a specific RGB value or corresponding hex value.

It's much easier and understandable to use "Pale Green" in code than using either (115,222,120) or 0x73de78


This project is an effort to provide a set of utility functions allowing the use of this classic RGB table for Clojure developers.

The functions may be used within the project or as an outside resource to simply look up the various color mappings.


The (most likely) useful functions are:

  • (get-hex-color colorname)
  • (get-rgb-color colorname)
  • (find-closest-color hexvalue)

For example:

(get-hex-color "Azure") ==> "f0ffff"

(get-rgb-color "beige") ==> (245 245 220)

(find-closest-color "30c0c0") ==> ("aquamarine")

For details, consult the API listings.
